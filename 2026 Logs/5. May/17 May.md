## Valid Parenthesis
```spoiler-markdown
Iterate over the string. 
If its an opening bracket, push it into the stack. 
If its a closing bracket, check if the stack is empty or if the top of the stack isn't the appropriate opening bracket for the current iterated character. Return false in either case. Note, you do not need to run a while loop to pop the stack! 
If neither occurs, pop the stack to remove the opening bracket. 
After all iterations, check if the stack is empty. If not, it means some opening brackets got left inside. 
```
